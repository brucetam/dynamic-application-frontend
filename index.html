Dynamic Form Component

Angular 動態表單元件 - 依據 JSON 配置自動生成表單

功能特色

  • 支援多種欄位類型：文字、文字區塊、文字編輯器、下拉選單、選項按鈕、核取方塊、檔案上傳等
  • 矩陣表單 (Matrix Field) 支援嵌套多層子表單
  • 依據 JSON 配置自動生成表單結構
  • 內建豐富的 Regex 驗證規則
  • 支援 Bootstrap 5 佈局，響應式設計
  • 支援拖曳檔案上傳
  • 檔案預覽功能
  • 自訂驗證規則與錯誤訊息
  • API 整合服務，支援表單提交
  • 相容主流桌面與手機螢幕

使用方式

1. 安裝依賴
# 使用 npm npm install bootstrap @angular/core @angular/forms @angular/common # 或使用 yarn yarn add bootstrap @angular/core @angular/forms @angular/common
2. 在模組中引入
import { NgModule } from '@angular/core'; import { BrowserModule } from '@angular/platform-browser'; import { ReactiveFormsModule } from '@angular/forms'; import { HttpClientModule } from '@angular/common/http'; import { DynamicFormComponent } from './components/dynamic-form/dynamic-form.component'; @NgModule({ declarations: [ AppComponent, DynamicFormComponent ], imports: [ BrowserModule, ReactiveFormsModule, HttpClientModule ], bootstrap: [AppComponent] }) export class AppModule { }
3. 在範本中使用
<!-- 基本用法 --> <app-dynamic-form [config]="formConfig" (formSubmitted)="onSubmit($event)" ></app-dynamic-form> <!-- 帶有初始資料 --> <app-dynamic-form [config]="formConfig" [initialData]="initialData" (formSubmitted)="onSubmit($event)" ></app-dynamic-form>
4. 定義表單配置
import { FormConfig } from './models/dynamic-form.model'; const formConfig: FormConfig = { id: 'user-form', title: '用戶註冊', apiEndpoint: '/api/users', method: 'POST', fields: [ { key: 'username', label: '用戶名', type: 'text', width: 6, validation: [ { type: 'required', message: '請輸入用戶名' } ] }, { key: 'email', label: '電子郵件', type: 'email', width: 6 } ] };
